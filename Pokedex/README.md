# Simple Pokedex

Projeto front-end para praticar consumo de API, manipulação do DOM e localStorage.

## Funcionalidades
- Busca de Pokémon por nome (via PokeAPI)
- Exibição de imagem, status (HP, Ataque, Defesa, Velocidade)
- Histórico das últimas 5 buscas (salvo no localStorage)
- Modo claro/escuro

## Tecnologias
- HTML5
- CSS3 (variáveis CSS para temas)
- JavaScript (Fetch API, DOM, localStorage)

## API utilizada
[PokeAPI](https://pokeapi.co/)
